  -F "name=Test User" \

  -F "phone=1234567890" \
  -F "dateOfBirth=1990-01-01" \
  -F "age=34" \
  -F "gender=Male" \
  -F "sscGpa=5.00" \
  -F "sscSchool=Test School" \
  -F "sscBoard=Test Board" \
  -F "hscGpa=5.00" \
  -F "hscCollege=Test College" \
  -F "hscBoard=Test Board" \
  -F "graduationSubject=Test Subject" \
  -F "graduationCgpa=3.50" \
  -F "graduationInstitution=Test University" \
  -F "skills=Skill1,Skill2,Skill3"

## Notes

- All dates should be in YYYY-MM-DD format
- Skills, languages, and hobbies should be comma-separated
- Previous jobs should be added one at a time using the "Add Previous Job" button
- Photo upload is optional but recommended for better CV appearance
- Phone numbers can be in any format
- GPA/CGPA can be out of 4.00 or 5.00 scale
